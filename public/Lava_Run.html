Time Saving

Dit eenvoudige Unity-script slaat de huidige tijd van een timer op wanneer de speler een trigger raakt (bijvoorbeeld bij het bereiken van de finish). Het gebruikt PlayerPrefs om de tijd (_currentTime) lokaal op te slaan, zodat deze later kan worden opgevraagd. Het script vereist een gekoppeld Timer-component dat de looptijd bijhoudt.
