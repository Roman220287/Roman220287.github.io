Rohn Flick

Dit was een groeps project met twee andere devs en drie artists. Voor dit project hadden we acht weken om een twinstick shooter te maken.

Ik heb in dit project vooral aan de core gameplay machanics gewerkt. Zoals: de slowmotion, de basic player movements en de granaat. Ook heb ik veel nieuwe dingen geleerd zoals hoe ik een granaat maak. Neem een kijkje hier onder naar een paar code snippits en de trailer van de game.

Pick-ups

Dit script zorgt ervoor dat een speler gezondheid of armor krijgt wanneer hij een object met de juiste tag oppakt (bijv. een medkit of armor pickup). Bij aanraking via een trigger wordt het corresponderende aantal punten toegevoegd aan de speler, en het object verdwijnt daarna uit de scène. Het script werkt samen met een PlayerHealth component waarin de waarden voor health en armor worden opgeslagen.

Throwing

Dit script laat de speler een beperkt aantal projectielen gooien vanuit een aangewezen punt. Bij het indrukken van een ingestelde toets wordt een object gespawned en met kracht vooruit én omhoog geworpen. Het gebruikt Rigidbody physics en een cooldown-systeem om herhaald gooien te beperken. Het script is volledig configureerbaar via de Unity Inspector.
